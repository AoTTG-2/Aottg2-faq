# Welcome to AoTTG 2 FAQ

Attack on Titan Tribute Game 2 (AoTTG 2) is a fan-made game currently in alpha development. As a voluntary project, we're constantly evolving and improving the game. This FAQ aims to address the most common questions and provide helpful information for our growing community.

## About AoTTG 2

AoTTG 2 is a tribute game based on the popular anime and manga series "Attack on Titan". It's a community-driven project, currently in its alpha stage, which means many features are still being developed and refined.
